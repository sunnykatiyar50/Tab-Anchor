Tab Anchor

Privacy Policy

Summary: Tab Anchor collects no personal data, sends nothing to any server, and stores everything locally in your browser only.

1. Overview

Tab Anchor ("the extension") is a browser extension for Google Chrome that allows users to lock tabs to a specific URL, rename tabs, and protect tabs from accidental navigation or closure. This Privacy Policy explains what data the extension handles and how.

2. Data We Do Not Collect

Tab Anchor does not collect, transmit, sell, or share any personal information. Specifically, we do not collect:

  • Browsing history or URLs you visit
  • Personal identifiers (name, email, IP address, device ID)
  • Usage analytics or telemetry
  • Crash reports or diagnostics
  • Any data that leaves your device

3. Data Stored Locally

The extension stores the following data in your browser's local storage (chrome.storage.local). This data never leaves your device:

  • Lock state — which tabs are locked and their locked URLs
  • Lock history — a log of past lock/unlock sessions (URL, timestamp, mode)
  • Custom tab names — any rename you apply to a tab
  • Settings — your theme and UI preferences

This data is stored per browser profile, is never synced to external servers, and can be cleared at any time from the extension's History page or via Chrome's built-in storage controls.

4. Network Requests

The extension makes no outbound network requests to any external server. The only fetch it performs is retrieving the favicon of the page currently open in your browser tab — this is a request to that page's own server (the site you are already visiting) solely to render a visual lock badge on the icon.

5. Permissions Explained

  • tabs — to read the current tab URL and detect navigation changes
  • tabGroups — to restore tab group membership when a locked tab is reopened
  • storage — to save lock state, history, and settings locally
  • contextMenus — to add right-click actions on the extension icon
  • <all_urls> — to inject the UI (warning banners, lock chip, rename dialog) into any tab the user locks

6. Third Parties

Tab Anchor does not integrate with any third-party services, analytics platforms, or advertising networks. No data is shared with any third party.

7. Children's Privacy

Tab Anchor does not knowingly collect any information from anyone, including children under the age of 13.
